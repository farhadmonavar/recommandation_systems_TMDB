# 🎬 Movie Recommender Systems

A comprehensive collection of Movie Recommendation System projects implemented in Python.

This repository covers the complete workflow of building recommendation systems:

- Understanding recommendation systems
- Data preprocessing and exploration
- Popularity-based recommendation
- Weighted average recommendation
- Hybrid recommendation systems
- Content-based filtering
- Feature engineering
- TF-IDF and Bag of Words
- Metadata creation
- Collaborative filtering
- K-Nearest Neighbors (KNN)
- Interactive recommendation systems

Core concepts:

- Content similarity
- Feature extraction
- Text preprocessing

Models include:

- Bag of Words
- TF-IDF
- Sigmoid Kernel
- Cosine Similarity
